## Open-source projects to read

- [**nanoGPT**](https://github.com/karpathy/nanoGPT) — minimum viable training.
- [**llm.c**](https://github.com/karpathy/llm.c) — GPT-2 in raw C.
- [**torchtitan**](https://github.com/pytorch/torchtitan) — PyTorch's clean distributed scaffold.
- [**Megatron-LM**](https://github.com/NVIDIA/Megatron-LM) — frontier-scale reference.
- [**OLMo**](https://github.com/allenai/OLMo) — fully open model + training stack.
- [**vLLM**](https://github.com/vllm-project/vllm) — production serving.
- [**SGLang**](https://github.com/sgl-project/sglang) — alternative production serving.
- [**llama.cpp**](https://github.com/ggml-org/llama.cpp) — CPU / consumer-GPU inference.
- [**TRL**](https://github.com/huggingface/trl) — SFT/DPO/PPO/GRPO reference.
- [**axolotl**](https://github.com/axolotl-ai-cloud/axolotl) — fine-tuning recipes.
- [**lm-evaluation-harness**](https://github.com/EleutherAI/lm-evaluation-harness) — benchmark plumbing.
- [**Inspect AI**](https://github.com/UKGovernmentBEIS/inspect_ai) — UK AISI eval framework.
- [**Unsloth**](https://github.com/unslothai/unsloth) — fast LoRA/QLoRA training.
- [**LangChain**](https://github.com/langchain-ai/langchain) — agent / RAG framework.
- [**LangGraph**](https://github.com/langchain-ai/langgraph) — agent state machines.
- [**Langfuse**](https://github.com/langfuse/langfuse) — open observability.

## How to keep current

- Daily: 15 minutes of arXiv-sanity / X / one newsletter.
- Weekly: read one paper deeply.
- Monthly: read one technical report (Llama, DeepSeek, Qwen, etc.) cover to cover.
- Quarterly: implement one paper from scratch.

This pays back at every chapter of every project.
