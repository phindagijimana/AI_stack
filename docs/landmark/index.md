# Landmark work

> Foundational papers, reference models, reference datasets, benchmarks, and books worth reading.

The list a senior research engineer is expected to know. Many of these are cited in detail in the technical chapters; this section gathers them as a reading list.

## Sub-sections

- **[Foundational papers](papers.md)** — the 50 papers that shaped modern LLM engineering.
- **[Reference models](models.md)** — the open-weights and closed-weights canon.
- **[Reference datasets](datasets.md)** — the corpora that shaped pretraining and post-training.
- **[Benchmarks](benchmarks.md)** — what each measures; pointers to leaderboards.
- **[Books worth reading](books.md)** — for depth that doesn't fit in a paper.

## How to use this section

If you're new: spend two weeks skimming the [foundational papers](papers.md). Pick five to read carefully and one to implement.

If you're mid-career: use this as a checklist. The papers and books here are the substrate of credible LLM engineering. Gaps are weaknesses.

If you're senior: use this as the canon you teach juniors from. Add your own annotations.
